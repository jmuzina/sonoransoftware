Sonoran CAD

Sonoran CAD is a state of the art, cross-platform, computer automated dispatching and records management system for gaming communities.

Sonoran CAD comes complete with real time connections, in-depth customization, voice commands, in-game integration, Discord WebHook notifications and more!

Features

Unique Window and Tab System

Sonoran CAD’s unique window and tab system allows you to multitask easier. Whether you’re viewing multiple records or referencing a list of 10-codes, our draggable and resizable window system allows you to manage your workflow efficiently.

Live Map Access

Easily link and display your community’s map or live map resource to be accessed by emergency services.

Record Management

Sonoran CAD’s record management system allows users to easily fill out in-depth reports for arrests, citations, warrants, BOLOs and more.

Sonoran CAD allows users to easily search records, licenses and vehicle registrations via character name or license plate. Voice commands also aid in faster searches with audio alerts for warning flags.

PDF Printed Records

Are you looking to view or file official criminal records? Only Sonoran CAD allows you to view and download a PDF copy!

Intuitive Search

Sonoran CAD allows users to easily search records, licenses and vehicle registrations via character name or license plate

Voice commands also aid in faster searches with audio alerts for warning flags.

Civillian Management

Civilians can easily add, edit, and remove characters, view criminal records, process DMV records, and more!

Civilians can also be automatically retrieved from your own in-game database without having to be manually created! For more information, check out our Database Sync functionality.

Community Branding

Branded Login Page

Sonoran CAD’s custom login screen allows you to display your community’s name, logo, and colors on your own domain!

Branded Emails

Sonoran CAD allows you to customize account creation and password recovery emails sent to your users with your logo, header, subheader, and body text!

Branded Loading Page

Sonoran CAD even allows you to customize the loading page icon for your community!

Customization

  • Custom 10-Codes

    Customize your community’s 10-codes and penal codes to be easily searched and referenced at any time.

  • Penal Codes

    Easily customize your penal codes to quickly reference and add charges to any criminal record report.

  • License Types

    Add, edit, and remove license types to be used in the DMV!

  • Page Colors

    Customize your community’s page colors!

  • Unit Status Codes

    Configure the unit status 10-Codes that your community uses!

Integration Options

Database Sync

Sonoran CAD’s Database Sync allows you to pull characters, licenses, and vehicle registrations all directly from your in-game database. This will work with nearly any MySQL Database:

  • FiveM (ESX or other)
  • Arma 3
  • Garry's Mod

and many more!

Discord Webhooks

Discord webhooks can be configured to alert your community when dispatches are sent, records are modified, unit’s are in a PANIC state, characters are modified, admin actions take place, and so much more!

Web API

Sonoran CAD’s web API allows you to send real time in-game data to your CAD! Automatically update unit location in real time, toggle a unit’s PANIC state in-game, send 911 calls, and more!
